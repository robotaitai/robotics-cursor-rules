# Portfolio Dashboards

Keep these lightweight. The goal is to help you navigate multiple project knowledge
folders without turning the portfolio root into another source of truth.

## Useful Dashboard Ideas

- active projects this week
- projects needing backfill or compaction
- projects with doctor warnings
- recent decisions across projects

## Pattern

- link to each project's `INDEX.md`
- optionally link to each project's `STATUS.md`
- summarize only what helps triage across projects
