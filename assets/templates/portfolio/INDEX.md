# Portfolio Knowledge Root

Open the shared knowledge root as a higher-level vault when you want to browse multiple
projects together, for example `~/agent-os/projects/`.

## What Lives Here

- one folder per project knowledge vault
- lightweight portfolio dashboards
- shared top-level notes about cross-project themes

## Suggested Use

- keep project knowledge authoritative inside each project folder
- use portfolio dashboards to link out to project roots, not to replace them
- treat cross-project rollups as navigation and orientation notes first
